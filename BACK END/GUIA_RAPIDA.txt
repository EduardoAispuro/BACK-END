═══════════════════════════════════════════════════════════════
  CÓMO SUBIR A GITHUB (RESUMEN)
═══════════════════════════════════════════════════════════════

1. Crea un nuevo repositorio en GitHub.com
   - Nombre sugerido: programacion-funcional-js
   - Público
   - Sin README inicial

2. Ejecuta estos comandos en la terminal:

   git remote add origin https://github.com/TU_USUARIO/programacion-funcional-js.git
   git branch -M main
   git push -u origin main

3. Activa GitHub Pages:
   - Settings → Pages
   - Source: main branch
   - Folder: / (root)
   - Save

4. Tu página estará en:
   https://TU_USUARIO.github.io/programacion-funcional-js/
